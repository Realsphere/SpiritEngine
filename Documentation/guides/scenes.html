Creating a Scene

With this guide you will get your first Scene going!

Step 1 - Creating a Scene object

To get started with a Scene, you first have to create a Scene object.

You can do this by creating an instance of the Scene struct.


Scene scene = new Scene();

Step 2 - Creating a Game Object

To add a GameObject to your scene, you have to create it.


GameObject gameObject = GameObject.CreateUsingMesh(StandarizedShapes.Cube, "Cube 1");
gameObject.HasCollision = false;
gameObject.HasGravity = false;
gameObject.Weight = 0f;
gameObject.Material = new SMaterial()
{
    Ambient = new SColor(255f, 255f, 255f, 255f),
    Diffuse = new SColor(255f, 255f, 255f, 255f),
    Emissive = new SColor(255f, 255f, 255f, 0f),
    Specular = new SColor(255f, 255f, 255f, 255f),
    SpecularPower = 20f
};

Okay, thats quite some code.

So what we're doing, is creating a GameObject with a Mesh (StandarizedShapes.Cube), with the name "Cube 1"

Then we remove Collision and Gravity from the Object.

Afterwards we make the Object static by setting its Weight to 0.

Then we set the Objects Material to a White material.

If we want texture on our cube, we can change the code to this:


GameObject gameObject = GameObject.CreateUsingMesh(StandarizedShapes.Cube, "Cube 1");
gameObject.HasCollision = false;
gameObject.HasGravity = false;
gameObject.Weight = 0f;
gameObject.Material = SMaterial.Create(STexture.Load("texture.png"));

This requires you to have a texture called "texture.png" in your build directory.

Step 3 - Adding a Game Object

Now to add our game object to the Scene, we can just call 

scene.GameObjects.Add(gameObject);

Step 4 - Setting the active Scene

We can set the active Scene like this: 


Game.ActiveScene = scene;

This will set the physics engine's Scene, and the renderer's Scene.
