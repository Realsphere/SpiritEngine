Getting Started

This guide will get you started with a Spirit project!

Step 1 - Download Spirit

This is the easiest Step, navigate over to the Spirit main page and click on the Download button.

Now a .ZIP downloaded, inside of this ZIP file, there is a file called "rsengine.dll", extract this file.

Step 2 - Create a new Project

We'll be using Visual Studio 2022 for all of our Guides, it is recommended you use it aswell.

Create a new .NET 6.0 C# Project, and add "rsengine.dll" as a reference.

Now you can use Spirit inside of your Project!

Step 3 - Using Spirit

To use Spirit you have to call Game.StartGame in your Main method.

Example of using this: 


Game.StartGame(new SpiritStartInfo()
{
    Company = "Realsphere Entertainment",
    CompanyLogo = "base 64 of your company logo",
    ShowFPS = true,
    FullScreen = false,
    GameIcon = "base 64 of your game icon",
    GameName = "Demo",
    Resolution = SpiritGameResolution.I4K
});
