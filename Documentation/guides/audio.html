Spirit Audio

Spirit includes an OpenAL and SndFile wrapper, and supports Spatial Audio.

Step 1 - Loading a WAV File

Loading a wave file in Spirit is pretty easy, this nice little one-liner loads a WAV file and creates an AudioSource: 

AudioSource source = new("sound.wav");

Pretty easy right? Well this line of code, loads a Wave File, and creates the OpenAL buffers.

All AudioSources have a Position, Volume, Pitch and a Velocity parameter.

The Listener position will always be the Player's position.

Step 2 - Playing the Audio Source

Now of course, if we want to hear something, we have to start the audio playback.

There are 4 methods regarding the Audio Playback: Play, PlayLooping, Stop, Pause

Play: Resumes / Plays the Audio Source.

PlayLooping: Loops the Audio Source.

Stop: Stops the Audio Source.

Pause: Pauses the Audio Source.
